Langthis Translation Widget allows any visitor to translate your site easily to about 55 Languarges. The button is added to the bottom of every post.

== Description ==
The free translation button for your website. The button allows your visitors to translate the page into about 55 Languarges. LangThis combines Google Translator, Live Translator, Babel Fish, InterTran, SDL and Promt into on button.

If you register for a LangThis account you can get statistics of translations.

Notice! The LangThis button, will place a link to LangThis.com at your website.

== Installation ==

1. Upload `langthis.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Edit your language and site id(if you have one) in the Settings page.

== Frequently Asked Questions ==

= Is Langthis free? =

Yes. The features you see today on Langthis will always be free. 

= Do I need to create an account? =

No. You only need to create an account if you want to see analytics on wich pages are being tranlated. The button will work just fine without a account.

= Is JavaScript required? =

Yes. JavaScript must be enabled. We load the actual interface via JavaScript at run-time, which allows us to upgrade the core functionality of the menu itself automatically everywhere. 

== PHP Version ==

PHP 5+ is preferred; PHP 4 is supported.
